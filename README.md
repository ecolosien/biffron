# biffron
Les Biffrons décrivent un phénomène mathématique où un tri pondéré de deux critères force 25% d'un groupe humain à se regrouper aux extrémités, créant une exclusion géométrique non intentionnelle. Il est interprété comme un essaimage évolutif (comme les abeilles), poussant les pro-sociaux à coloniser de nouvelles niches.
